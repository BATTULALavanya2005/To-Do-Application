# 📝 My To-Do List App

A responsive and user-friendly To-Do List application built using HTML, CSS, JavaScript, jQuery, and Bootstrap.

## 🚀 Features

- Add tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Pending / Completed)
- Local Storage support
- Responsive UI

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 4
